# Larngear Registration

15th Larngear Registration Website

## Installation

1. Set up `.env` variable. Change `.env.sample` to `.env`
2. `npm install && gulp`
3. `php composer.phar install`
4. Done

PS. you can also deploy via Heroku (`Procfile` available)
